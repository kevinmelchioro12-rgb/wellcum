# VelociTAI — Accertamento velocità che regge in giudizio

**Per Comandi di Polizia Locale ed Enti.** Software di accertamento automatico
dei limiti di velocità (Art. 142 CdS) che copre l'intero ciclo e — soprattutto —
produce **verbali difendibili**, riducendo annullamenti e ricorsi.

## Il problema
Troppi verbali da autovelox vengono **annullati o vinti in ricorso** per vizi
formali, prove deboli o errori di calcolo. Ogni annullamento è costo, contenzioso
e perdita di credibilità.

## La soluzione
Rilevamento veicolo → misura velocità → lettura targa (ANPR) → **prova con catena
di custodia** → calcolo sanzione a norma → verbale → notifica PEC. Tutto
tracciato, verificabile, a prova di contestazione.

## Perché VelociTAI (differenziatori)
- **Verbale difendibile**: tolleranza di legge (5% / min 5 km/h), riferimenti
  normativi puntuali, termini perentori, **prova firmata (HMAC) verificabile** e
  **registro di audit a prova di manomissione**.
- **Sicurezza & GDPR by-design**: dati personali protetti e redatti, accessi
  controllati, segreti fuori dal codice.
- **Affidabilità**: isolamento dei guasti, auto-riparazione, **niente doppie
  multe** (idempotenza). Comando di **auto-diagnosi** integrato.
- **Nessun lock-in**: backend e gestionali intercambiabili, API aperte.

## Cosa vede l'operatore
Console web con esiti per veicolo, verbali emessi e stato del sistema; API JSON
per l'integrazione col gestionale del Comando.
